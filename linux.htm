Core Linux Administrator Skills

1. Linux Fundamentals

  • Understanding Linux distributions (Ubuntu, RedHat, CentOS, Debian)
  • File system structure and hierarchy
  • Basic commands (ls, cd, mv, cp, rm, cat, grep, find)
  • File permissions and ownership
  • Working with users and groups

2. System Administration

  • User and group administration
  • Managing system processes
  • System startup and services management (systemd)
  • Managing environment variables
  • Scheduling tasks with cron and at

3. Storage & File Systems

  • Partitioning disks
  • Linux file systems (ext4, xfs, btrfs)
  • LVM (Logical Volume Manager)
  • Mounting and unmounting drives
  • NFS and SAMBA shares

4. Networking

  • Configuring network interfaces
  • Understanding TCP/IP concepts
  • Hostname and DNS configuration
  • SSH setup and management
  • Firewall management (iptables, firewalld, ufw)

5. Package Management

  • APT package manager (Debian/Ubuntu)
  • YUM/DNF package manager (CentOS/RedHat)
  • Installing, removing, and updating packages
  • Repository configuration

6. Security & Hardening

  • User authentication & password policies
  • SSH key management
  • SELinux/AppArmor basics
  • System audit logs (journald, syslog)
  • Fail2ban, antivirus, firewalls

7. Scripting & Automation

  • Shell scripting (Bash)
  • Automating administrative tasks
  • Using environment variables & conditions
  • Running scripts on schedule using cron

8. Virtualization & Containers

  • KVM virtualization
  • Docker container management
  • Basic Kubernetes skills (optional)
  • Working with cloud-init on VMs

9. Monitoring & Troubleshooting

  • Using top, htop, vmstat, iostat
  • Log analysis in /var/log
  • Network troubleshooting tools (ping, traceroute, netstat)
  • Disk and memory issue diagnosis
