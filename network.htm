Network Fundamentals Skills

1. Networking Basics

  • Understanding OSI and TCP/IP models
  • IP addressing (IPv4 & IPv6)
  • Subnetting and supernetting
  • MAC addresses and ARP
  • Port numbers and protocols (TCP/UDP)

2. Network Devices & Components

  • Routers, switches, hubs, and access points
  • Firewalls and network security appliances
  • Network interface cards (NICs)
  • Cables (Ethernet, fiber) and connectors
  • Patch panels and structured cabling

3. Routing & Switching

  • Static and dynamic routing
  • VLANs and inter-VLAN routing
  • Routing protocols (RIP, OSPF, EIGRP, BGP basics)
  • Switching concepts: STP, trunking, port channels

4. Network Services & Protocols

  • DHCP, DNS, NAT, and PAT
  • HTTP, HTTPS, FTP, SMTP, POP3, IMAP
  • VPNs (IPSec, SSL VPN)
  • SNMP for network monitoring

5. Wireless Networking

  • Wi-Fi standards (802.11a/b/g/n/ac/ax)
  • Wireless security (WEP, WPA, WPA2, WPA3)
  • SSID, channels, and frequency bands
  • Access point configuration and roaming

6. Network Security Fundamentals

  • Firewalls and ACLs
  • VPNs and secure tunneling
  • Intrusion detection/prevention (IDS/IPS)
  • Encryption & authentication basics
  • Common network attacks and mitigation

7. Troubleshooting & Monitoring

  • Ping, traceroute, netstat, nslookup, dig
  • Wireshark packet analysis
  • Network performance monitoring
  • Identifying and resolving connectivity issues

8. Cloud & Modern Networking

  • Basic understanding of cloud networking (AWS, Azure, GCP)
  • SDN (Software Defined Networking) concepts
  • Load balancing and high availability
  • IPv6 deployment in modern networks
